시스템 아키텍처 구성도
1. 백엔드 및 데이터 (Backend & Data Layer)
이 시스템의 핵심 로직과 데이터 처리를 담당하는 영역입니다.

FastAPI (ECR/ECS)

시스템의 메인 서버 역할을 하며, 아래 구성 요소들과 연결됩니다.

DB (RDS): 관계형 데이터베이스와 연동되어 데이터를 저장 및 관리합니다.

LLM Model API: 거대 언어 모델(LLM) API와 연동됩니다.

ML Model (S3): S3에 저장된 머신러닝 모델을 활용합니다.

2. 프론트엔드 (Frontend Layer)
FastAPI 서버와 통신하며 사용자 및 관리자에게 인터페이스를 제공합니다. 두 웹 애플리케이션 모두 S3와 CloudFront를 통해 배포됩니다.

User React (사용자 웹)

일반 사용자를 위한 인터페이스입니다.

주요 기능:

소비 분석

이상 거래

맞춤 광고

AI 리포트

Admin React (관리자 웹)

시스템 관리자를 위한 인터페이스입니다.

주요 기능:

트래픽 현황

군집 리포트
